# Pacote basico (WebPack)

### Requisitos locais
* node (http://nodejs.org)
* yarn (yarnpkg.com)

### Instalação
* `$ cd rockerseat-es6`
* `$ yarn install`

### Rodando
* `$ yarn dev` 

### Anotação
Conforme ocorre a mudança no arquivo `app.js` o `parse-env` do `Babel` se responsabiliza em reescreve o arquivo `bundle.js` para a interpretação dos navegadores.
